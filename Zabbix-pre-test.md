## Задание предварительное zabbix

1. На хосте **gw**. Создайте ssh-ключ (тип ed_25519) для пользователя student.

2. Разместите публичную часть созданного ключа на хостах **ws**, **infra**, **proxy**, **server**, **db**, **gw** в учетной записи пользователя student для подключения без пароля.

3. На хосте gw. Внесите изменения в файл `/etc/hosts`, добавьте следующие строки:

```ini
10.0.10.220 ws
10.0.10.100 infra
10.0.10.50 proxy
10.0.20.10 server
10.0.20.3 db
10.0.20.1 gw
```

4. На хосте gw. Установите пакет `https://repo.zabbix.com/zabbix/7.0/rhel/9/x86_64/zabbix-release-latest-7.0.el9.noarch.rpm`

5. На хосте **gw**. Добавьте правило для firewalld, разрешающее входящие пакеты 80/tcp

6. На хосте **gw**. Установите сервис httpd и добавьте его в автозагрузку 

7. На хосте **gw**. Скопируйте из журнала `/var/log/messages` все строки, принадлежащие сервису httpd в файл `/home/student/check/httpd.otchet`
