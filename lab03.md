
## Модуль 3: Режимы работы и преобразование данных 

**Задание: Настройка активного и пассивного режима агента Zabbix.**

---
**План:**  

- Настройте агент в активном режиме. 
- Настройте агент в пассивном режиме. 
- Преобразуйте данные с использованием предварительной обработки. 
- Настройте интервалы сбора данных. 
- Изучите полученные данные. 

---

### Практическая работа

### 1. Настройка агента в активном режиме: 

1. На сервере **zabbix-db** откройте конфигурационный файл агента Zabbix. 
```bash
sudo nvim /etc/zabbix/zabbix_agent2.conf
``` 

2. Установите параметр `ServerActive` на адрес вашего Zabbix сервера. 
```ini
Server=127.0.0.1
ServerActive=10.0.20.10
RefreshActiveChecks=60
ForceActiveChecksOnStart=1
```

3. Перезапустите службу агента для применения изменений. 
```bash
sudo systemctl restart zabbix-agent2
```

### 2. Настройка агента в пассивном режиме: 

1. На сервере **zabbix-db** откройте конфигурационный файл агента Zabbix. 
```bash
sudo nvim /etc/zabbix/zabbix_agent2.conf
``` 

2. Установите параметр `Server` на адрес вашего Zabbix сервера, а параметр `ServerActive` закомментируйте. 
```ini
Server=10.0.20.10
#ServerActive=10.0.20.10
```

3. Перезапустите службу агента для применения изменений. 
```bash
sudo systemctl restart zabbix-agent2
```

### 3. Преобразование данных с использованием предварительной обработки: 

1. Перейдите в интерфейс Zabbix и откройте **Сбор данных -> Узлы сети**. 

2. В строке **zabbix-db** выберите **элемент данных**.

3. В поле **имя** напишите **ping** и **Применить**.  

4. Найдите строку **Linux by Zabbix agent: Zabbix agent ping** и нажмите на **Zabbix agent ping**

5. В открывшемся окне найдите **Родительские элементы данных Linux by Zabbix agent** и нажмите на **Linux by Zabbix agent**
6. Во вкладке **Предобработка** нажмите добавить и заполните поля:
   - Имя: Замена
   - Параметры, строка поиска : 1
   - Параметры, замена: Up
7. Перейдите в интерфейс Zabbix и откройте **Мониторинг -> Последние Данные**. В поле **Имя** должно быть указано **Zabbix agent ping**, тогда в столбце **Последние данные** будет указано **`Up(1)`** 

### 4. Настройка интервалов сбора данных: 
1. Перейдите в интерфейс Zabbix и откройте **Сбор данных -> Узлы сети**. 

2. В строке **zabbix-db** выберите **элемент данных**.

3. В тегах найдите Student:familia и нажмите
4. Выберите ранее созданный элемент Check uptime via SSH
5. В поле Интервал обновления укажите 3m, затем Обновить
6. Повторите шаги 1-4 и нажмите **последние данные** - вы должны увидеть что столбец **последняя проверка** будет некоторое время изменяться до 3-х минут а потом сброситься на значение от 1 до 59 секунд, а поле **Последнее значение** увеличится на три минуты.
